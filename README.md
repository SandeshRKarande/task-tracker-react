# Task Tracker (React + Vite)

A simple Task Tracker application built using **React** and **Vite**.

## Features

- Add new tasks
- Display tasks in a list
- Delete tasks
- Press **Enter** to add tasks
- Tasks persist using **LocalStorage**

## Tech Stack

- React
- Vite
- JavaScript
- HTML/CSS
